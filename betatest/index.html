Meteomat - Probabilistic Weather Forecasts (Testing Page)

Interactive weather dashboard showing ECMWF ensemble forecasts for Europe. Visualizes forecast uncertainty bands so you see confidence ranges, not just single predictions.

  • ECMWF ensemble model data via Open-Meteo
  • Probabilistic uncertainty visualization
  • Coverage across Europe
  • Built with Streamlit and Plotly
Data: ECMWF via Open-Meteo | Geocoding: OpenStreetMap
